VECL-TTS:Voice Identity and Emotional Style Controllable Cross-Lingual Text-to-Speech Synthesis

Model Architecture
VAW-GAN-EVC [1]: A state-of-the-art VAW-GAN-based EVC framework. The decoder is conditioned with F0 and one-hot emotion ID.
EVC-USEP (proposed) : The proposed one-to-many controllable VAW-GAN-based EVC framework as descript in Figure 1.
w/o VDP : EVC-USEP without the proposed Virtual Domain Pairing. For more details please refer to the paper.
w/o FPM : EVC-USEP without the proposed Fake Pair Masking. For more details please refer to the paper.
Note:
(1) EVC-USEP is an one-to-many conversion model for all three EVC tasks, while VAW-GAN-EVC is one-to-one model and trained three times for each task resepectively.
